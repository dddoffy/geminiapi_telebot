# Gemini API TelegramBot

This is a small telegram bot that uses GeminiAPI, giving you an opportunity to chat with Gemini using your telegram client.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install requirements.txt
```

## Usage
To use this bot, you should :
1. GeminiAPI key [here.](https://aistudio.google.com/api-keys)
2. Create a Telegram bot and api key through your Telegram client.
3. Create an env file using your API keys as shown in an example.
4. Run the main.py file and write a question to your bot.
